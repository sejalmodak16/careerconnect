# CareerConnectAI

CareerConnectAI is an AI-powered platform that helps job seekers match their resumes with relevant job opportunities. It analyzes resume content and compares it against job listings to suggest the best-fit roles, making the job search process faster and more targeted.

## Features

- 📄 Resume upload and parsing
- 🤖 AI-driven resume-to-job matching
- 🎯 Personalized job recommendations based on skills and experience
- 🔍 Search and browse job listings
- 📊 Match score/insights to show how well a resume fits a job

## Getting Started

### Prerequisites
- Node.js installed on your machine
- npm

### Installation

1. Clone the repository
```bash
   git clone https://github.com/sejalmodak16/careerconnect.git
   cd careerconnect
```

2. Install dependencies
```bash
   npm install
```

3. Start the development server
```bash
   npm run dev
```

## Usage

1. Upload your resume
2. Browse or search available job listings
3. View match scores and personalized recommendations
4. Apply to jobs that best fit your profile
